Language of Your Choice (Loyc) will be a wide-ranging project to improve interoperability between programming languages, to make new programming languages easier to implement, to make existing programming languages easier to extend, to help people convert source code between languages, and to define a cross-language standard library.

Loyc will be a collection of FOSS tools...

  • for language conversion (transforming source code between different languages)
  • for cross-language library programming, also known as acmeism
  • for IDE tools (code completion lists, various kinds of code visualization, intellisense)
  • for code analysis and transformation
LLLPG and Enhanced C# are also included in the Loyc project.

Please visit the Main Documentation (doc comments available upon request) or the SourceForge project page.
