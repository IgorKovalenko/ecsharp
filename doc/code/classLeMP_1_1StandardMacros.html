static LNode LeMP.StandardMacros.KeyNameComponentOf ( LNode  name)
inlinestatic

Retrieves the "key" name component for the nameof(...) macro.

The key name component of global::Foo!int.Bar!T(x) (in C# notation global::Foo<int>.Bar<T>(x)) is Bar. This example tree has the structure ((((global::Foo)!int).Bar)!T)(x)).

References Loyc.Syntax.LNode.Args, Loyc.Syntax.CodeSymbols.Dot, Loyc.Syntax.CodeSymbols.Of, and Loyc.Syntax.LNode.Target.

inlinestatic

Searches a list of expressions/statements for one or more patterns, and performs replacements.

Parameters
stmtsA list of expressions/statements in which to search.
patternsEach pair consists of (A) something to search for and (B) a replacement expression. Part A can use the substitution operator with an identifier inside (e.g. $Foo) to "capture" any subexpression, and part B can use the same substitution (e.g. $Foo) to insert the captured subexpression(s) into the output.
replacementCountNumber of replacements that occurred.
Returns
The result of applying the replacements.

LNodeExt.MatchesPattern is used for matching.
