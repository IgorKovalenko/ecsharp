override void Loyc.Collections.Impl.CPBitArrayLeaf< T >.MoveFirst ( CPEnumerator< T >  e)
inlinevirtual

(1) Add()s an Entry to e.Stack pointing to the first (lowest) item in the node, using e.Key.Offset as the value of Entry.KeyOffset; (2) extracts the key to e.Key such that e.Key.Offset+e.Key.Left is the length of the complete key (if e.Key.Buffer is too small, it is copied to a larger buffer as needed); (3) if the current item points to a child, this method advances e.Key to the end of the key so that e.Key.Left is 0, and calls MoveFirst on the child; (4) otherwise, this method leaves e.Key.Offset equal to e.Stack.Last.KeyOffset, so that e.Key.Left is the number of bytes of the key that are stored in this node.

override bool Loyc.Collections.Impl.CPBitArrayLeaf< T >.MoveNext ( CPEnumerator< T >  e)
inlinevirtual

Moves to the next item in the node.

Returns
Returns true if the next item was extracted or false if the end of the node was reached.

Upon entry to this method, e.Stack.Last.Node == this and e.Stack.Last.KeyOffset == e.Key.Offset.

This method (1) increases e.Stack.Last.Index to point to the next item in the node; (2) returns false if Index has advanced past the last item in the node; (3) otherwise, repeats steps (2)-(4) in the documentation of MoveFirst() and returns true.
