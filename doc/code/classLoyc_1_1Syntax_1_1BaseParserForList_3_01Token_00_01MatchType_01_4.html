Remarks

An base class designed for parsers that use LLLPG (Loyc LL(k) Parser Generator) and receive tokens from any IEnumerator{Token}.

This base class for LLLPG parsers reads tokens from IList{Token}, but you can also pass an IEnumerable{Token} or IEnumerator{Token} to the constructor and it will use BufferedSequence{T} to convert the sequence to a list.
